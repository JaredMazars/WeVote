## 🚀 Quick Start

1. **Click** "Set AGM Timer" button
2. **Enter** times:
   - Start: `12:04` (when voting begins)
   - End: `13:05` (when voting ends)
3. **Click** "Set Timer"
4. **Done!** Timer appears in header for everyone
