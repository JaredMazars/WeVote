# WeVote vs Lumi Global - Competitive Analysis

## Date: December 7, 2025

---

## Executive Summary

This document provides a comprehensive competitive analysis between **WeVote** (Forvis Mazars voting platform) and **Lumi Global** (established AGM/voting platform provider). The analysis identifies gaps, strengths, and strategic recommendations to position WeVote as a superior solution.

---

## 1. Lumi Global - Strengths Identified

### 1.1 Meeting Management Module
- **Pre-AGM Setup**: Configure agenda, resolutions, voting requirements
- **Attendee Management**: Track who's joined, proxy assignments
- **Document Distribution**: Share meeting materials, annual reports
- **Live Meeting Control**: Chair controls, moderator features

### 1.2 Multi-Channel Access
- **Web Platform**: Browser-based access
- **Mobile Apps**: iOS and Android native apps
- **SMS Integration**: Vote via text message
- **Telephone Voting**: Call-in voting for accessibility

### 1.3 Enhanced Security
- **Two-Factor Authentication (2FA)**: SMS/Email verification
- **Blockchain Verification**: Immutable vote records
- **Encrypted Transmission**: End-to-end encryption
- **Audit Trail**: Complete vote history with timestamps

### 1.4 Real-Time Features
- **Live Results Display**: Instant vote tallying
- **Q&A Module**: Submit questions to board during meeting
- **Live Polling**: Ad-hoc polls during meeting
- **Participant Chat**: Attendee communication

### 1.5 Advanced Analytics
- **Voting Patterns**: Trend analysis over multiple meetings
- **Participation Rates**: Track engagement metrics
- **Proxy Utilization**: Analyze proxy voting behavior
- **Export Options**: PDF, Excel, custom reports

### 1.6 Compliance & Governance
- **Regulatory Compliance**: SEC, ASX, UK Companies House standards
- **Legal Templates**: Pre-approved resolution templates
- **Vote Verification**: Shareholder eligibility checks
- **Audit Reports**: Compliance documentation

### 1.7 Multi-Language Support
- **Interface Localization**: 20+ languages
- **Real-time Translation**: Meeting materials translation
- **Regional Formats**: Date, time, currency formatting

### 1.8 Integration Capabilities
- **Calendar Integration**: Outlook, Google Calendar
- **CRM Integration**: Salesforce, HubSpot
- **Video Conferencing**: Zoom, Teams, Webex
- **Share Registry**: Direct integration with registrars

- ✅ **Auto-Balance Calculation**: Automatic discretionary vote allocation

## 4. Strategic Recommendations to Surpass Lumi

### Phase 1: Critical Enhancements (Next 30 days)

#### 1. Two-Factor Authentication
**Why**: Security is non-negotiable for voting platforms
**Implementation**:
- SMS-based 2FA using Twilio
- Email verification on new device login
- "Trust this device" option
- Emergency backup codes

#### 2. Email Notification System
**Why**: Users need meeting reminders and vote confirmations
**Implementation**:
- SendGrid/Mailgun integration
- Templates: Meeting invite, vote confirmation, proxy assignment, results
- Scheduled reminders (24hr, 1hr before AGM)
- Unsubscribe management

#### 3. Enhanced PDF/Excel Reporting
**Why**: Corporate clients need formal documentation
**Implementation**:
- PDF generation using jsPDF or Puppeteer
- Excel export using SheetJS
- Report templates: Vote results, audit logs, proxy summary
- Branded report headers

#### 4. Progressive Web App (PWA)
**Why**: Mobile access without app store approval
**Implementation**:
- Service worker for offline capability
- Add to home screen prompt
- Push notifications
- Mobile-optimized UI
